Central Bank of the Russian Federation (cbr.ru) as currency exchange rate sync
provider for Commerce multicurrency module.

DEPENDENCIES
Commerce Multicurrency provider for RUB depends on the Commerce multicurrency 
module.

INSTALLATION
Install the module as usual.

CONFIGURATION
1. Select "Central Bank of the Russian Federation" as sync provider on currency
   conversion settings page: admin/commerce/config/currency/conversion
2. Run cron or sync manually to synchronize the rates.
